Sou um **Engenheiro de Software** com domínio em **Java, Go, Kotlin e TypeScript**, especializado no desenvolvimento Backend, arquiteturas limpa e microsserviços e integração entre sistemas.

Tenho experiência com Spring Boot, Quarkus e Node.js para construção de APIs e serviços backend, mensageria com Apache Kafka, criptografia de dados em runtime, autenticação JWT, bancos de dados relacionais, MapReduce com Hadoop, e práticas de DevOps com Docker e CI/CD.

## 🧠 Sobre mim

- ☕ **Java + Spring Boot / Quarkus** — APIs REST, microsserviços, Clean Architecture, Spring Security e JWT
- 🐿️ **Go** — serviços backend com foco em performance e baixo acoplamento
- 🟦 **TypeScript / Node.js** — APIs, serviços assíncronos e aplicação de princípios SOLID
- 🎯 **Kotlin** — desenvolvimento backend
- 📨 **Apache Kafka** — mensageria assíncrona com producer/consumer em ambientes containerizados
- 🔐 **Segurança** — criptografia de dados sensíveis em runtime, autenticação JWT e controle de acesso por roles (RBAC)
- 🗄️ **PostgreSQL, MySQL e MongoDB** — modelagem relacional e não relacional, ORM (JPA/Hibernate), migrations e otimização de queries
- 🐳 **Docker + CI/CD** — aplicações containerizadas, pipelines automatizados e ambientes reproduzíveis
- 🧪 **Testes** — JUnit, Mockito e TDD aplicados em projetos reais
- 📊 **Big Data** — processamento distribuído com MapReduce e Hadoop
- 🤖 **Integrações com IA** — consumo de APIs de IA para funcionalidades inteligentes em sistemas
